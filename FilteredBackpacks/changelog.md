## [0.1.1] - 2026-01-10
### Fixed
- FilteredBackpacks didn't load assets on non-Windows systems because of the release zip file structure.
- Papyrus couldn't be put into the agriculture backpack by default.
- One of base game mining bag recipes wasn't disabled and showed an error.

## [0.1.0] - 2025-07-05
### Added
- Agriculture, forestry and mining bags with simple and sturdy variants.
### Removed
- Base game mining bag (both variants) for easier code maintenance.
